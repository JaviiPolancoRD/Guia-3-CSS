Concepto Descripción Concepto Descripción
Tabla en HTML Representa datos en dos o más dimensiones. Formulario en HTML Permite la recopilación y envío de datos del usuario a un servidor.
Etiqueta table Organiza datos como textos, imágenes y enlaces en filas y columnas. Atributos de table Modifican la apariencia y el comportamiento de la tabla, como border, cellspacing, y width.
Etiqueta tr Define una fila de una tabla en HTML. Atributo tr Incluyen align, valign, bgcolor, entre otros.
Etiqueta td y th Td: define una celda en una fila de una tabla. Th: define una celda de encabezado. Atributos td y th Permiten ajustar características como colspan, align, width, entre otros.
Tablas avanzadas Son tablas que usan etiquetas adicionales como thead y tfoot para mejorar la estructura. Etiqueta thead Agrupa el contenido del encabezado de la tabla.
Etiqueta tfoot Agrupa el contenido de pie de la tabla. Etiqueta tbody Agrupa el contenido principal o cuerpo de la tabla.
Etiqueta form Es el contenedor de un formulario y especifica dónde y cómo se envían los datos del formulario. Etiqueta input Crea diferentes tipos de campos de entrada, como texto, contraseña, botones, etc.
Etiqueta label Asocia un texto descriptivo con un elemento de formulario. Etiqueta select Crea una lista desplegable con opciones.
Etiqueta textarea Permite introducir texto en múltiples líneas dentro de un formulario. Etiqueta fieldset, legend Fieldset: Agrupa varios elementos de un formulario. Legend: Proporciona un título para el grupo.
Etiqueta meter Representa una medida escalar dentro de un rango reconocido. Etiqueta progress Muestra el progreso de una tarea en curso.
Etiqueta datalist Proporciona una lista de opciones predefinidas. Etiqueta summary Se utiliza con details para crear un widget desplegable.

Contenedor (wrapper)

El contenedor es un elemento envolvente que engloba todo el contenido de una página web. Su función principal es centralizar y controlar el diseño, creando límites o márgenes en la disposición visual.

Cabecera (header)

El header es la parte superior de una página web, y normalmente incluye el logotipo, el nombre del sitio, el título o el eslogan, así como elementos de navegación primarios.

Contenido (content)

El contenido es la parte principal de una página web, donde se presenta la información más relevante para el usuario.

Menú (menu)

El menú o barra de navegación facilita la exploración del sitio, conteniendo enlaces a diferentes secciones o páginas.

Pie (footer)

El footer es la parte inferior de una página web, y generalmente contiene enlaces adicionales como políticas de privacidad, información de contacto, o enlaces a redes sociales.

Lateral (sidebar)

La barra lateral es una sección que suele colocarse a un lado del contenido principal, mostrando información secundaria o complementaria.

Etiqueta div

El elemento div en HTML es un contenedor genérico utilizado para agrupar bloques de contenido o elementos.
